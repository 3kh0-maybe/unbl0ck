# Project-Hub
This project is meant to serve as a hub for any game (web or ports) to be run whilst not being detected by an extensions blocking capabilities.

## Features
- Extensive Libray of Games
- Multiple Emulators
- Unblocked Games
- Tab Favicon Changer
- Tab Title Changer
- Constantly Updating
- Open-Source
